---
## 1. Name
- Variables/Functions:
  (Hump Method)
  For example:
  ```javascript
  function wsjsgTestFunction() {
    let wsjsgTestVariable = 1;
  }
  ```

- Classes:
  (Pascal Method)
  For example:
  ```javascript
  class WsjsgTestClass {}
  ```

- Constants:
  (JConstants Method)
  For example:
  ```javascript
  const WSJSG_TEST_CONSTANT = 1;
  ```
